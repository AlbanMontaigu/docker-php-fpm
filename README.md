# docker-nginx-php-plus

## Introduction

This image is an addition to my [nginx-php](https://hub.docker.com/r/amontaigu/nginx-php/) image.

**Php modules added :**
* iconv
* mcrypt
* mbstring
* mysqli
* pdo_mysql
* zip
* gettext
* exif
